### Requirements

- asgiref==3.7.2
- django==4.2
- faker==20.1.0
- pillow==10.1.0
- python-dateutil==2.8.2
- six==1.16.0

Use conda to install the requirements.txt with: 

```conda install --file requirements.txt```

Otherwise use:

```pip install -r requirements.txt```
